

# Project - Free Levoit Air Purifiers

Collection of custom ESPHome firmware and hardware projects for Levoit air purifiers, eliminating cloud dependency and enabling native Home Assistant integration.

* [Generic Levoit ESPHome Component](./components/levoit/), Custom ESPHome component for Levoit Air Purifiers.

[List of supported Devices](./devices/README.md) – Overview of all supported and tested devices
* [Levoit Core 200s](./devices/levoit-core200s) – Custom ESPHome Firmware 
* [Levoit Core 300s](./devices/levoit-core300s) – Custom ESPHome Firmware
* [Levoit Core 400s](./devices/levoit-core400s) – Custom ESPHome Firmware
* [Levoit Vital 100s](./devices/levoit-vital100s) – Custom ESPHome Firmware

Soon:
* [Levoit Core 600s](./devices/levoit-core600s) – Custom ESPHome Firmware - WIP
* [Levoit Vital 200S](./devices/levoit-vital200s) – Custom ESPHome Firmware - WIP

Special:
* [Levoit LV PUR 131s](./devices/levoit-lv131s/) – Custom Firmware + MCU & sensor upgrade + hardware hack
* [Levoit Mini](./devices/levoit-mini) – Custom PCB, 3D parts, hardware hack
