# Levoit esphome component for Core and Vital Series 

This component supports the following model 

* Levoit Vital 100s, MCU 1.0.5
* Levoit Vital 200s
* Levoit Core 300s, MCU 2.0.11
* Levoit Core 400s, MCU 3.0.0

## Features

* Home Assistant Fan, with speeds and Auto Modes as presets
* Setting Timer + show set time and remaining time, like 5h 33 min remaining 
* Setting Auto Mode + Room Size
* Setting Display, Light Detect, Child Lock
